그래서 대화를 잘 못하나봐 나도 아직 어려워요 어떻게 해야되는 건지 모르겠어 정답이라는 게 있는 걸까 싶기도해

그니까 내가 뭔 말을 하고싶은 거냐면........... 그냥 고맙다구요 그리고 내가 많이 싫어하는 거로 느낀다면 오해라구 말해주고 싶어요
